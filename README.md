# AutoFold v3

AutoFold는 다운로드 폴더를 자동으로 정리해주는 파이썬 프로그램이다.  
파일 확장자 기준으로 분류하며, 동일한 이름의 파일이 존재할 경우 자동으로 이름을 변경하여 덮어쓰기를 방지한다.  
실수 시 원래대로 되돌리는 기능 또한 존재한다.

---

## 프로젝트 목적

본 프로젝트는 파이썬을 처음 학습하며 파일 자동화와 폴더 관리 개념을 익히기 위해 제작되었다.

---

## 주요 기능

- 확장자 기반 자동 폴더 정리
- 정리 전 미리보기 (Dry Run)
- 한 번 실행 / 자동 반복 실행 선택
- 파일 이동 기록 로그 저장
- 되돌리기 (Undo) 기능
- 중복 파일명 자동 처리
- 이미 정리된 폴더는 재정리하지 않음

---

## 정리 기준

기본 설정은 다음과 같습니다:

| 폴더 | 확장자 |
|----|----|
| Images | `.png`, `.jpg`, `.jpeg` |
| Videos | `.mp4`, `.mov` |
| Documents | `.pdf`, `.docx`, `.txt` |
| Hancom | `.hwp` |
| Zips | `.zip`, `.rar` |
| Others | 그 외 모든 파일 |

※ 설정은 `config.py`에서 자유롭게 수정할 수 있습니다.

---

## 실행 방법

### 1. Python으로 실행하기

```bash
python main.py
```

실행 후 모드를 선택할 수 있습니다:

```
1. 미리보기
2. 한 번만 정리
3. 계속 자동 정리
```

---

### 2. exe 파일로 실행 (빌드된 경우)

- `AutoFold.exe` 더블 클릭  
- Python 설치 필요 없음

---

## 되돌리는 방법

```bash
python undo.py
```

- 마지막 정리 기록을 기준으로 **한 번만** 되돌릴 수 있습니다  
- 되돌린 후 로그는 초기화됩니다

---

## exe 빌드 방법 (개발자용)

```bash
pip install pyinstaller
pyinstaller --onefile --name AutoFold main.py
```

결과물:

```text
dist/AutoFold.exe
```

---

## 주의사항

- Downloads 폴더 기준으로 동작합니다  
- 자동 정리 모드는 5초 주기로 실행됩니다  
- 중요한 파일이 있다면 **미리보기 모드 사용을 권장합니다**
